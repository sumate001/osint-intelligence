# OSINT//DESK — Intelligence Cycle Completion
## 6 modules ที่เติมให้ครบมาตรฐานงานข่าวกรอง

เวอร์ชัน 1.0 · มิถุนายน 2568

## ภาพรวม

ระบบเดิมแข็งแรงด้าน collection และ analysis แต่ขาดชั้นที่ทำให้เป็น intelligence operation เต็มรูปแบบ เอกสารนี้เพิ่ม 6 module เพื่อปิดช่องว่างตาม intelligence cycle มาตรฐาน

```
[1. Requirements/PIR] → กำหนดโจทย์
        ↓
[Collection] ← มีอยู่แล้ว (adapters, OSINT, dark web)
        ↓
[2. Source Reliability] → ประเมินความน่าเชื่อถือ
        ↓
[Processing/Analysis] ← มีอยู่แล้ว (triage, investigation)
        ↓
[3. Collaboration] → หลาย analyst ทำร่วมกัน
[4. Analytic Confidence] → ระบุความมั่นใจ + dissent
[5. Deception Detection] → ตรวจ disinformation
        ↓
[Dissemination] ← มีอยู่แล้ว (Brief Builder)
        ↓
[6. Knowledge Base] → institutional memory ข้ามเคส
```

## Module 1: Requirements & Tasking (PIR)

### หลักการ

งานข่าวกรองที่ดีเริ่มจากคำถามว่า "เราต้องการรู้อะไร" ไม่ใช่รอข้อมูลไหลเข้ามา Priority Intelligence Requirements (PIR) คือโจทย์ที่ Intelligence Lead กำหนด แล้วระบบจัดสรรทรัพยากรไปหาคำตอบ

### ความสามารถ

Intelligence Lead สร้าง PIR ระบุคำถามหลัก เกณฑ์ความสำเร็จ และ deadline ระบบแตก PIR เป็น collection tasks ย่อย กระจายไปยัง adapters, watchlist และ analyst ที่เกี่ยวข้อง จากนั้นติดตามว่าแต่ละ PIR มีข้อมูลตอบแล้วกี่เปอร์เซ็นต์

เมื่อข้อมูลใหม่เข้ามา ระบบจับคู่อัตโนมัติว่าตอบ PIR ข้อใด และแจ้งเตือนเมื่อ PIR ใดใกล้ตอบครบหรือเลย deadline

### โครงสร้างข้อมูล

PIR ประกอบด้วยคำถามหลัก คำถามย่อย (Essential Elements of Information) แหล่งที่คาดว่าจะตอบได้ ผู้รับผิดชอบ ระดับความสำคัญ และสถานะความคืบหน้า

## Module 2: Source Reliability Scoring

### หลักการ

งานข่าวกรองใช้ระบบให้คะแนนแหล่งข้อมูลแบบ NATO Admiralty Code แยกสองมิติ คือความน่าเชื่อถือของแหล่ง (A ถึง F) และความน่าเชื่อถือของข้อมูล (1 ถึง 6) ทำให้ประเมินได้ว่าข้อมูลชิ้นนี้เชื่อได้แค่ไหน

### ความสามารถ

ทุกแหล่งข้อมูลได้เรตติ้งความน่าเชื่อถือที่ปรับตามประวัติ แหล่งที่เคยให้ข้อมูลถูกต้องบ่อยได้คะแนนสูงขึ้น แหล่งที่เคยให้ข้อมูลผิดถูกลดคะแนน ระบบติดตามอัตโนมัติว่าข้อมูลจากแต่ละแหล่งสุดท้ายแล้ว verified หรือ false

เมื่อ analyst เห็นข้อมูล จะเห็นเรตติ้งแหล่งทันที เช่น B2 หมายถึงแหล่งค่อนข้างเชื่อถือได้และข้อมูลน่าจะจริง ช่วยให้ตัดสินใจได้เร็วว่าควรให้น้ำหนักแค่ไหน

### มาตราส่วน

ความน่าเชื่อถือของแหล่ง: A เชื่อถือได้สมบูรณ์ B เชื่อถือได้ปกติ C ค่อนข้างเชื่อถือได้ D ไม่ค่อยเชื่อถือ E ไม่น่าเชื่อถือ F ประเมินไม่ได้

ความน่าเชื่อถือของข้อมูล: 1 ยืนยันจากแหล่งอื่น 2 น่าจะจริง 3 อาจจะจริง 4 น่าสงสัย 5 ไม่น่าเป็นไปได้ 6 ประเมินไม่ได้

## Module 3: Collaboration & Multi-Analyst

### หลักการ

งานสืบสวนจริงหลายคนทำด้วยกัน ต้องมีระบบที่ให้ analyst หลายคนทำงานบนเคสเดียวกันโดยไม่ชนกัน เห็นว่าใครทำอะไร และส่งต่องานได้

### ความสามารถ

แต่ละเคสมี activity feed แสดงว่าใครเพิ่มหลักฐาน แก้ไขอะไร เมื่อไหร่ Analyst comment บน evidence แต่ละชิ้นได้ ตั้งคำถามหรือโต้แย้งได้ เมื่อต้องส่งต่องาน มี handoff ที่บันทึกบริบทให้คนรับทำต่อได้ทันที

เมื่อ analyst สองคนเห็นไม่ตรงกันเรื่องการตีความหลักฐาน ระบบบันทึกทั้งสองมุมมองไว้ ไม่บังคับให้เลือกข้างก่อนเวลา ซึ่งสำคัญมากในงานข่าวกรองเพราะ premature consensus คือกับดักที่ทำให้พลาด

### Real-time

หลาย analyst เปิดเคสเดียวกันพร้อมกันได้ เห็น cursor และการแก้ไขของกันและกันแบบ real-time ป้องกันการทำงานซ้ำซ้อน

## Module 4: Analytic Confidence & Dissent

### หลักการ

Brief ระดับมืออาชีพต้องระบุระดับความมั่นใจของแต่ละข้อสรุป ไม่ใช่แค่บอกว่า verified หรือไม่ และต้องเปิดพื้นที่ให้ dissenting opinion เพราะความเห็นที่แตกต่างมักเป็นสัญญาณเตือนที่มีค่า

### ความสามารถ

ทุกข้อสรุปใน Brief ระบุระดับความมั่นใจสามระดับ คือมั่นใจสูง มั่นใจปานกลาง และมั่นใจต่ำ พร้อมเหตุผลว่าทำไม ระดับความมั่นใจคำนวณจากคุณภาพหลักฐาน จำนวนแหล่งที่ยืนยัน และเรตติ้งความน่าเชื่อถือของแหล่ง

เมื่อทีมเห็นไม่ตรงกัน analyst บันทึก dissenting view ได้ พร้อมเหตุผล Brief สุดท้ายแสดงทั้งข้อสรุปหลักและความเห็นแย้ง ทำให้ผู้อ่านเห็นภาพครบและตัดสินใจเองได้

### Structured Analytic Techniques

ระบบรองรับเทคนิควิเคราะห์ที่เป็นระบบ เช่น Analysis of Competing Hypotheses ที่ให้ analyst ลิสต์สมมติฐานที่เป็นไปได้ทั้งหมด แล้วประเมินว่าหลักฐานแต่ละชิ้นสนับสนุนหรือขัดแย้งกับสมมติฐานใด ช่วยลด confirmation bias

## Module 5: Deception Detection

### หลักการ

โดยเฉพาะงานข่าว ต้องระวังข้อมูลที่จงใจปล่อยมาเพื่อหลอก ไม่ว่าจะเป็น disinformation, ข้อมูลที่ปล่อยมาเพื่อ discredit คู่แข่ง หรือ honeypot ที่ล่อให้รายงานข่าวผิด

### ความสามารถ

ระบบประเมินทุกข้อมูลสำคัญด้วยคำถาม counter-intelligence ใครได้ประโยชน์ถ้าเรารายงานข่าวนี้ ข้อมูลนี้มาในจังหวะที่น่าสงสัยหรือไม่ แหล่งมีแรงจูงใจซ่อนเร้นไหม รูปแบบการเผยแพร่เป็นธรรมชาติหรือมีการจัดฉาก

เมื่อพบสัญญาณที่น่าสงสัย ระบบ flag ไว้และเตือน analyst ให้ตรวจสอบเพิ่มก่อนรายงาน โดยเฉพาะข้อมูลที่ดูสมบูรณ์แบบเกินไปหรือมาในจังหวะที่เป็นประโยชน์ต่อฝ่ายใดฝ่ายหนึ่งอย่างชัดเจน

### Bot Network Detection

สำหรับข้อมูลจากโซเชียลมีเดีย ระบบวิเคราะห์ว่าการแพร่กระจายเป็นธรรมชาติหรือถูกขับเคลื่อนด้วย bot network โดยดูรูปแบบเวลาการโพสต์ ความคล้ายของข้อความ และเครือข่ายบัญชีที่เกี่ยวข้อง

## Module 6: Knowledge Base & Institutional Memory

### หลักการ

เคสที่ปิดแล้วควรกลายเป็นความรู้ที่ค้นได้ เพื่อให้การสืบสวนใหม่ใช้ประโยชน์จากงานเก่า คำถามอย่าง "บริษัทนี้เคยปรากฏในเคสไหนบ้าง" หรือ "บุคคลนี้เชื่อมโยงกับใครในเคสก่อนหน้า" ต้องตอบได้ทันที

### ความสามารถ

ทุกเอนทิตีที่เคยปรากฏในทุกเคสถูกเก็บใน knowledge graph กลาง เมื่อเริ่มเคสใหม่ ระบบดึงประวัติของทุกเอนทิตีที่เกี่ยวข้องมาแสดงทันที ว่าเคยปรากฏที่ไหน เชื่อมโยงกับใคร และผลการสืบสวนเดิมเป็นอย่างไร

Analyst ค้นหาข้ามทุกเคสได้ เช่นค้นว่ามีเคสไหนเกี่ยวข้องกับการประมูลภาครัฐบ้าง หรือบุคคลใดปรากฏในหลายเคส ทำให้เห็นรูปแบบที่ข้ามเคสเดียวมองไม่เห็น

### Cross-Case Pattern Detection

ระบบตรวจจับรูปแบบที่ปรากฏซ้ำข้ามเคส เช่นบริษัทที่ใช้รูปแบบการตั้งบริษัทแบบเดียวกัน หรือเครือข่ายบุคคลที่ปรากฏร่วมกันบ่อยผิดปกติ ซึ่งอาจชี้ถึงเครือข่ายที่ใหญ่กว่าที่เห็นในเคสเดียว

## การเชื่อมโยงทั้งระบบ

6 module นี้ไม่ได้แยกจากกัน แต่เสริม module เดิม

PIR กำหนดโจทย์ให้ทั้งระบบ Source Reliability ติดตามทุกข้อมูลที่เข้ามา Collaboration ทำงานบนทุกเคส Analytic Confidence ปรากฏในทุก Brief Deception Detection ตรวจทุกข้อมูลสำคัญ และ Knowledge Base เก็บทุกอย่างไว้ใช้ต่อ

ผลคือระบบที่ทำงานครบ intelligence cycle ตั้งแต่กำหนดโจทย์ เก็บข้อมูล ประเมินความน่าเชื่อถือ วิเคราะห์ร่วมกัน ระบุความมั่นใจ ตรวจการหลอกลวง นำเสนอ และเก็บเป็นความรู้ — ไม่ใช่แค่เครื่องมือ แต่เป็นระบบข่าวกรองที่สมบูรณ์
